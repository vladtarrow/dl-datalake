# dl-datalake Frontend

The web-based dashboard for the DL Datalake system. Built with **React**, **TypeScript**, and **Vite**.

## Features

- **Dashboard**: Table view of all stored datasets with search and filtering.
- **Ingestion Wizard**: User-friendly form to download data from CCXT exchanges.
- **Data Preview**: High-performance Paginated table for reviewing Parquet data segments.
- **Feature Store**: GUI for uploading and managing custom data features and versions.
- **Dark Mode Support**: Styled with **Tailwind CSS**.
